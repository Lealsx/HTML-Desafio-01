O que seria os Servidores

(Voltar)

Acho que, quando pensamos em servidores, pensamos naquelas coisas gigantes que só são vistas em filmes e séries e estão totalmente longe de nosso alcance. Infelizmente, é verdade, mas bem próximos de nós temos os servidores de games. Claramente, os servidores 'reais' são muito melhores em tudo e são feitos tanto para armazenar suas senhas e seu usuário, como um banco de dados, quanto para hospedagem de sites. Como dito nas minhas aulas, o servidor não somente recebe algo, ele também envia as coisas aos usuários. Mas isso gera um problema, que é: como vamos enviar as coisas para as pessoas, sendo que meu computador entende de uma forma e o seu de outra? Como enviamos as coisas? Devido a pensamentos assim, foi criada uma tabela chamada de ASCII. Ela nada mais é do que uma tabela global para que todos os computadores entendam tudo da mesma forma. Os servidores nada mais são do que máquinas 'escaláveis' (quer dizer que podem ser aprimoradas ao longo dos avanços tecnológicos), e máquinas que podem tanto armazenar coisas quanto enviar coisas ao seu usuário ou cliente

O que são Tags

(Voltar)

As tags em HTML são elementos fundamentais que definem a estrutura e o conteúdo de uma página da web. Elas são usadas para marcar um texto em itálico ou em negrito. Com as tags, podemos formatar um texto, adicionar títulos e subtítulos, entre outras coisas. Sua estrutura básica é composta por pares, com uma tag de abertura e outra de fechamento.

  • Tag de abertura: Inicia o elemento e contém o nome da tag entre os sinais de menor e maior (< >).
  • Tag de fechamento: Termina o elemento e é semelhante à tag de abertura, mas com uma barra (/) antes do nome da tag.

Estrutura Básica do HTML

Voltar

Uma estrutura básica do HTML (HyperText Markup Language) segue uma organização padrão que inclui os principais elementos necessários para criar uma página web. alguns exemplos abaixo:

  1. tag HTML: envolve todo o conteúdo da página.
  2. tag HEAD: é um lugar onde colocamos o nome do nosso titulo que aparecera no navegador
  3. Tag Body: onde é contido todo o conteudo do visivel da pagina
